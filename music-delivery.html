Musicity

A music delivery app.

A simple app that delivers content from a backend on request. Users have to sign up and be logged in to be able to gain access to the resources.

This app is built upon a Node js/express server and a mongoDb database to store the songs' details and user information. The front end is built using the pug templating engine and core CSS(no bootstrap or any other library). To see more, follow the link: https://musicity.onrender.com/
